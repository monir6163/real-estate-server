## 🎯 ডেটা-সচেতনতা পরীক্ষা

### টেস্ট 1: এজেন্ট (প্রপার্টি লিস্টার)

**প্রস্তুতি:**

1. একটি এজেন্ট অ্যাকাউন্টে লগইন করুন
2. কয়েকটি প্রপার্টি লিস্ট করুন (কমপক্ষে 2টি)

**প্রশ্ন করুন:**

```
"আমার কাছে কতটা প্রপার্টি আছে?"
```

**প্রত্যাশিত প্রতিক্রিয়া:**
এজেন্টদের জন্য, চ্যাটবট বলবে:

```
"আপনার 3টি সম্পত্তি রয়েছে। এটি সত্যিই লুকানো -
আপনার সাম্প্রতিক তালিকা অন্তর্ভুক্ত:
  • [Property 1 Name] - [Location] - ₹[Price]
  • [Property 2 Name] - [Location] - ₹[Price]
  • [Property 3 Name] - [Location] - ₹[Price]"
```

---

### টেস্ট 2: কাস্টমার (ক্রেতা)

**প্রস্তুতি:**

1. একটি কাস্টমার অ্যাকাউন্টে লগইন করুন
2. কয়েকটি প্রপার্টি বুক করুন অথবা রিভিউ দিন

**প্রশ্ন করুন:**

```
"আমার বুকিংস কী?"
```

**প্রত্যাশিত প্রতিক্রিয়া:**

```
"আপনার [X] সক্রিয় বুকিং রয়েছে। আপনার সম্প্রতি
বুকিংগুলি দেখায় যে আপনি সম্পত্তি বুকিং অভিজ্ঞতা
করছেন অথবা সম্প্রতি সম্পম্ন করেছেন।

আপনি যদি কোনও বুকিং বাতিল করতে চান বা
বিস্তারিত চান তবে আমাকে অবহিত করুন।"
```

---

### টেস্ট 3: প্রসঙ্গ-সচেতন প্রতিক্রিয়া

**প্রশ্ন করুন:**

```
"আমার প্রপার্টিজ কীভাবে প্রচার করব?"  [এজেন্টদের জন্য]
```

**প্রত্যাশিত প্রতিক্রিয়া:**
চ্যাটবট উল্লেখ করবে:

- আপনার বর্তমান প্রপার্টিজের সংখ্যা
- প্রচারের পরামর্শ
- বিশেষভাবে আপনার প্রপার্টিজের জন্য

## 💻 API টেস্টিং (কমান্ড লাইন)

### Curl দিয়ে সরাসরি টেস্ট করুন

```bash
# প্রথমে একটি টোকেন পান
curl -X POST http://localhost:5000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"agent@example.com","password":"123456"}'

# সেই টোকেন দিয়ে চ্যাট সার্ভিস টেস্ট করুন
curl -X POST http://localhost:5000/api/v1/chat/stream \
  -H "Content-Type: application/json" \
  -H "Cookie: token=YOUR_TOKEN_HERE" \
  -d '{
    "messages": [
      {"role": "user", "content": "আমার কাছে কতটা প্রপার্টি আছে?"}
    ]
  }'
```

## 🐛 সমস্যা সমাধান

### সমস্যা 1: 401 Unauthorized

**কারণ:** আপনি লগইন করেননি

**সমাধান:**

1. প্রথমে লগইন করুন
2. পরে চ্যাটবট ব্যবহার করুন

---

### সমস্যা 2: চ্যাট উত্তর দিচ্ছে না

**লগ চেক করুন:**

```bash
# ব্যাকএন্ড টার্মিনালে ত্রুটি আছে কিনা দেখুন
# "Chat streaming error" বা "API Error" লুকান
```

**সমাধান:**

1. OPENAI_API_KEY সেট আছে কিনা চেক করুন
2. API কী বৈধ কিনা যাচাই করুন
3. OpenAI অ্যাকাউন্টে ব্যালেন্স আছে কিনা চেক করুন

---

### সমস্যা 3: ডেটা দেখাচ্ছে না

**লক্ষণ:** চ্যাটবট "No specific user data available" বলে

**কারণ:** ব্যবহারকারীর কাছে কোনো প্রপার্টি/বুকিং নেই

**সমাধান:**

1. এজেন্ট হলে কয়েকটি প্রপার্টি তৈরি করুন
2. কাস্টমার হলে একটি বুকিং করুন
3. তারপর চ্যাট করুন

## 📋 সম্পূর্ণ টেস্ট স্ক্রিপ্ট

```typescript
// test.ts - নিয়ে একটি টেস্ট ফাইল তৈরি করুন
import axios from "axios";

const test = async () => {
  // 1. লগইন
  const login = await axios.post("http://localhost:5000/api/v1/auth/login", {
    email: "agent@example.com",
    password: "123456",
  });

  const token = login.data.data.token;

  // 2. চ্যাট পাঠান
  const stream = await axios.post(
    "http://localhost:5000/api/v1/chat/stream",
    {
      messages: [{ role: "user", content: "আমার প্রপার্টিজ বলুন" }],
    },
    {
      headers: {
        Cookie: `token=${token}`,
      },
      responseType: "stream",
    },
  );

  // 3. প্রতিক্রিয়া প্রিন্ট করুন
  stream.data.on("data", (chunk) => {
    console.log(chunk.toString());
  });
};

test();
```
